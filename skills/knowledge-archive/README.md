# knowledge-archive

Post-work habit check: after finishing a feature or change, scan what was done and decide whether anything is worth adding to the repository knowledge base.

Most features produce nothing worth archiving. This skill catches the ones that do.

## Install

```bash
npx skills add deweyou/skills --filter knowledge-archive
```

## Usage

```
/knowledge-archive
```

Run after merging a PR or finishing a feature. Works best alongside `code-knowledge`.
